# MaratonUSP

O MaratonUSP é um grupo de pogramação competitiva sediado no Instituto de Matemática e Estatística (IME), na USP.

# Este repos

Esse repositório busca demonstrar as minhas soluções para problemas propostos assim como manter arquivados os meus péssimos programas desenvolvidos.
